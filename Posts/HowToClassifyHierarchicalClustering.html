How to classify your illustrations using Hierarchical Clustering


Hierarchical Clustering is a simple, intuitive yet powerful method in Machine Learning that allows you to build models that recoignizes similar patterns on a dataset of image
from sklearn.cluster import AgglomerativeClustering 

	from scipy.cluster.hierarchy import dendrogram, linkage
	import matplotlib.pyplot as plt
	Z = linkage(features_list, method='ward')

	# Dendogram
	plt.figure(figsize=(10, 7))
	dendrogram(Z)
	plt.show()

	# Hierarchical Clustering
	cluster = AgglomerativeClustering(n_clusters=5, affinity='euclidean', linkage='ward')
	cluster_labels = cluster.fit_predict(features_list)
